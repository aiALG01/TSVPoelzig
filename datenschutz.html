2. Hosting

Diese Website wird als statische Seite bei Netlify (Netlify, Inc.) gehostet. Beim Aufruf der Seite verarbeitet Netlify automatisch technische Zugriffsdaten (u. a. IP-Adresse, Datum und Uhrzeit des Zugriffs, aufgerufene Seite), wie es bei jedem Website-Hosting technisch notwendig ist. Es werden keine eigenen serverseitigen Datenbanken mit Besucherdaten betrieben. Weitere Informationen: Datenschutzerklärung von Netlify.

3. Schriftarten

Diese Website nutzt die Schriftarten Clash Display und Synonym. Beide werden lokal von diesem Server ausgeliefert (Self-Hosting), es findet keine Verbindung zu Google Fonts, Fontshare oder anderen Schriftarten-Anbietern beim Seitenaufruf statt. Es werden dabei keine Daten an Dritte übertragen.

4. Cookies und Consent

Für den technischen Betrieb der Seite selbst werden keine Cookies gesetzt. Einzelne erweiterte Inhalte, zum Beispiel eine eingebettete Kartenansicht auf der Seite „Spiel- und Trainingsorte“, laden externe Inhalte nach. Diese werden erst eingebunden, nachdem aktiv zugestimmt wurde (Klick auf „Karte laden“ bzw. „Erweiterte Inhalte erlauben“ im Consent-Banner). Ohne Zustimmung findet keine Verbindung zum jeweiligen Anbieter statt. Die Zustimmung wird lokal im Browser gespeichert (localStorage) und kann jederzeit durch Löschen der Browserdaten zurückgesetzt werden.

4.1 Kartendarstellung (OpenStreetMap)

Nach Zustimmung wird eine Karte von OpenStreetMap eingebunden. Dabei kann die IP-Adresse an die OpenStreetMap Foundation übertragen werden. Weitere Informationen: Datenschutzerklärung der OpenStreetMap Foundation.

5. Neuigkeiten-Verwaltung (/admin)

Der Vorstand pflegt Neuigkeiten über einen passwortgeschützten Verwaltungsbereich, der nur berechtigten Vereinsmitgliedern zugänglich ist. Für Besucherinnen und Besucher der Website ohne Zugang zu diesem Bereich findet dabei keine Datenverarbeitung statt.

6. Kontaktformular

Das Kontaktformular auf der Seite „Kontakt“ wird über Netlify Forms verarbeitet, eine Funktion des Hosting-Anbieters Netlify. Die eingegebenen Daten (Name, E-Mail-Adresse, Nachricht) werden dabei an Netlify übertragen und von dort per E-Mail an den 1. Vorsitzenden weitergeleitet. Eine Datenverarbeitung findet ausschließlich zur Bearbeitung der jeweiligen Anfrage statt. Weitere Informationen: Datenschutzerklärung von Netlify.
